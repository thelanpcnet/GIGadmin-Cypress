# Introduction
This document will help you to run GIGadmin Cypress scripts properly
## Requirements
You have to install:
- NodeJS
- Yarn
- Cypress

You also need my env file to run these scripts
## Step by step guide
After forking this repository, access to its directory then run Cypress
```
cd GIGadmin-Cypress
yarn cypress
```
